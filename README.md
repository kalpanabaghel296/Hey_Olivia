# Hey_Olivia
This repository will contain all the code part and documentation of the Windows AI voice assistant named Olivia. 

# The flow of Olivia is:

User Voice
   ↓
Microphone
   ↓
Speech-to-Text (SpeechRecognition)
   ↓
Text Preprocessing (NLP)
   ↓
ML Intent Classifier
   ↓
Action Executor (Web)
   ↓
Text Response
   ↓
Text-to-Speech
   ↓
User hears response


# The Tech Stack we used is :

Language: Python
Speech-to-Text: SpeechRecognition (Google API)
NLP: NLTK / spaCy
ML: Scikit-learn
Backend: Flask
Frontend: HTML, CSS, JavaScript
Text-to-Speech: pyttsx3
Version Control: Git + GitHub
